## Good capture rules

- Use seeded/demo data so the visuals feel consistent across the portfolio.
- Prefer one clean desktop screenshot per major flow before adding mobile variants.
- Avoid screenshots with browser bookmarks, devtools, or temporary localhost warnings visible.
- Keep admin screenshots focused on the workspace, filters, or editing tools instead of full-page browser chrome.
